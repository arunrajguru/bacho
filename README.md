# bacho
This is a premium landing page + scam detection interface.

🔹 Purpose

A web app called “Bacho — Know Before You Go”
👉 It helps users detect scams (messages, emails, etc.)

🧩 Key Sections
1. 🎨 UI/Design System
Dark theme with Apple-style UI
Custom fonts (DM Sans, Noto Sans Devanagari)
Smooth animations & gradients
Custom cursor effects
Glassmorphism (blur + transparency)
2. 🚨 Risk Alert System
#alarmLayer → red flashing background for high-risk scams
#alarmBar → animated warning bar on top
Triggered when scam risk = HIGH
3. 🧭 Navigation Bar
Brand name (Devanagari + Latin mix)
Links + CTA button
Changes color based on risk level
4. 🏠 Hero Section
Big headline + animated gradient text
Tagline: “Know Before You Go”
Stats display (like trust metrics)
CTA buttons (Scan / Try)
5. 🔍 Scam Detector UI (Core Feature)

This is the main functionality:

Text area input → user pastes message/email
Sample scam examples (clickable pills)
“Scan” button
6. 🤖 AI Thinking Animation

When scanning:

Progress % shown
Steps like:
analyzing text
checking patterns
verifying risk
Animated orb + progress bar
7. 📊 Results Section

After scan:

Risk level:
🔴 High
🟠 Medium
🟢 Safe
Visual gauge (circle meter)
Risk explanation
